Conteúdo aprendido até aqui

Desde a primeira aula até esta, vimos o que é IDE, HTML e CSS e como fazer uma estrutura básica de HTML, como head, body e etc. Depois disso, vimos conceitos básicos e avançados de formatação e estilização de texto, como texto sublinhado, itálico, negrito, superior ou inferior a linha de texto padrão, cor de texto e de fundo, e etc.

Vimos também como inserir vídeos e imagem à página web, incluindo texto alternativos caso a imagem "quebre" ou seja removida do ar. Fazendo isso, deixamos nossa página web pronta para problemas eventuais, ficando mais profissional.
